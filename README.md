# Fischfuetterung

Statische Web-App zur Verwaltung von Chargen, Becken, Fuetterung, Mortalitaet, Sortierung und Schlachtung.

Die App speichert Daten lokal im Browser per `localStorage`. Es ist kein Backend erforderlich.
